Repo Rescue Desk Proof Note

Repo Rescue Desk is the current CAM_CAM showpiece: a read-only triage layer that turns a repo universe into risks, capability clusters, ranked opportunities, and graph artifacts before CAM-PULSE or any AI agent mutates code.

What This Proves

  • CAM_CAM can scan a real repo universe.
  • It classifies repos into capability clusters.
  • It identifies governance concerns before acting.
  • It ranks useful app and showpiece opportunities.
  • It exports dashboard, graph, GraphRAG, Logseq, Markmap, and Freeplane artifacts.
  • It leaves source repos untouched.

Run

git clone https://github.com/deesatzed/CAM_CAM.git
cd CAM_CAM
PYTHONPATH=apps/repo_rescue_desk python -m repo_rescue_desk.cli \
  --root /path/to/folder/containing/git/repos \
  --out-dir tmp/repo_rescue_desk/latest

Outputs

  • repo_inventory.json
  • repo_rescue_dashboard.html
  • opportunity_rankings.md
  • risk_report.md
  • repo_mindmap_markmap.md
  • repo_mindmap_freeplane.mm
  • repo_graph.json
  • graphrag_context.md
  • logseq/pages/*.md

Verification

PYTHONPATH=apps/repo_rescue_desk python -m pytest apps/repo_rescue_desk/tests -q
./scripts/test_repo_rescue_desk_showpiece.sh
python -m html.parser docs/cam_cam_showpiece.html
git diff --check

The harness creates a controlled fixture repo universe, runs the CLI, and verifies dashboard, inventory, GraphRAG, graph JSON, risk report, opportunities, Logseq, Freeplane, and Markmap artifacts.

Limits

The first version uses deterministic README, file, and git metadata analysis. It does not yet perform embedding-based semantic clustering or ingest the results into CAM's durable learning memory. Those are the next useful upgrades.
